API

How to programmatically utilize our API.

/api/packages/{name}/{version} (GET)

Returns a single JSON object for the specific version of the package.

/api/packages/{name} (GET)

Returns an array of JSON objects for all versions of the package in descending order.

/api/packages (GET)

Returns an array of JSON objects for all packages and all versions, grouped by package.

/api/packages/{name}/{version}/download (GET)

Direct download link to the package ZIP contents.
